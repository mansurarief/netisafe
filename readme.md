### Installation

1. Clone the repo
   ```sh
   git clone git@github.com:mansurarief/safesocial.git
   ```

2. Go into the repo
   ```sh
   cd safesocial
   ```

3. Install requirements
   ```sh
   pip install -r requirements.txt
   ```

4. Run the webserver
   ```sh
   phyton webserver.py
   ```
5. Visit the rendered page at: [http://localhost:5000/](http://localhost:5000/)
